Welcome to the project's page.

I'm attempting to develop a API for the TeamSpeak 3 Server using the ServerQuery it provides via telnet, implementing it in Python 3.

And yes, I am aware that this has been done already and it's even on GitHub but I'm doing this as a practice project and maybe I'll get to use it later on my own server.
